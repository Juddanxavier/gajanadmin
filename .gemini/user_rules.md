# User Rules

These are project-specific rules that should always be followed:

## Package Manager

- **Always use npm** - Never use pnpm, yarn, or other package managers
- Use `npm install` for installing dependencies
- Use `npm run` for running scripts
- Use `npx` for running one-off commands

## Framework & Libraries

- Use shadcn/ui for UI components
- Use Supabase for backend services

## General Guidelines

- Follow Next.js best practices
- Maintain consistent code style
- Keep components modular and reusable
